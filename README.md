# Comparative-Oncology-Analytics
인간 유방암 데이터와 반려견 림프종 데이터의 통합적 접근
. 연구 배경 및 목적
•	배경: 헬스케어 및 의료 분석에서 머신러닝의 중요성이 대두되고 있으며, 특히 '비교 종양학(Comparative Oncology)'은 인간과 동물의 암 발생 기전 및 특징이 유사하다는 점에 착안하여 양측 모두에게 이익이 되는 진단/치료법을 연구하는 유망 분야임.
•	문제 인식: 인간의 암 데이터는 비교적 풍부하고 정제되어 있으나, 수의학 분야의 동물 암 데이터는 접근이 어렵고(data-poor), 비정형적이며 복잡함.
•	목적:
1.	데이터가 풍부한 인간 유방암 데이터를 활용하여 '세포의 형태학적 특징' 기반의 악성/양성 분류 모델(rf model,logistic)의 유효성을 검증함 (Project 2).
2.	'비교 종양학' 관점을 적용하여, 실제 동물 데이터(반려견 림프종)의 복잡성을 다루고, 암 발병에 영향을 미치는 핵심 '위험 요인(Risk Factors)'을 통계적으로 분석하고 해석함 (Project 1).
